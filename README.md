# LostArk Clone

## 프로젝트 소개
로스트아크 스타일의 탑다운 액션 RPG 프로젝트입니다.

## 개발 환경
- Unreal Engine 5.4
- Visual Studio 2022
- C++

## 주요 시스템
- 클릭 이동
- 몬스터의 플레이어 추적

## 구현 예정 시스템
- 크리스탈 소환에 의한 몬스터 소환 시스템
- 노말 몬스터와 레어 몬스터 클래스 구분
- 애님 클래스 작성

## 실행 방법
1. `LostArk.uproject` 실행
2. 필요 시 Visual Studio에서 `Development Editor` 빌드

## 폴더 구조
Source/
├── Player/

└── UI/
