# LostArk Clone

## 프로젝트 소개
로스트아크 스타일의 탑다운 액션 RPG 프로젝트입니다.

## 개발 환경
- Unreal Engine 5.4
- Visual Studio 2022
- C++

## 주요 시스템
- 클릭 이동
- 몬스터 AI
- 스킬 시스템
- 카오스 던전

## 실행 방법
1. `.uproject` 실행
2. Visual Studio Generate
3. Build Development Editor
